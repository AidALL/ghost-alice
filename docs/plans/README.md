# Ghost-ALICE OS Planning Policy

Language: 🇺🇸 English | [🇰🇷 한국어](../ko/plans/README.md)

This directory is reserved for public planning documents. Do not use it for
local work queues, private integration notes, customer-specific roadmaps, or
scratch handoff state.

## Public Planning Rules

- Track contributor-facing work in GitHub Issues or public roadmap documents.
- Keep private deployment plans outside this repository.
- Do not preserve completed-history notes here; use Git history, releases, and
  merged pull requests for that.
- Treat `.tmp/` and local notebooks as scratch only. Promote only public,
  contributor-relevant decisions into tracked docs.

## Current Public Roadmap

No public roadmap items are tracked in this directory for the initial release.
